# 🧪 Deep Dives & Fundamentals

Ce dépôt regroupe mes explorations techniques, mes implémentations "from scratch" et mes recherches architecturales.

## 📂 Structure du Repo

- **`fundamentals/`** : Comprendre le "comment ça marche" en réimplémentant des outils complexes (Signals, Promises, Servers).
- **`architectures/`** : Études sur la combinaison de frameworks, le routing, la gestion de données et le scaling.
- **`prototypes/`** : Tests rapides de bibliothèques, de nouvelles APIs ou de concepts isolés.

Chaque section est divisée par langage (`typescript/`, `go/`, etc.).
